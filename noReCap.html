noReCap

Take the frustration out of captcha with a browser extension based on Buster.

Building upon Buster, noReCap completely automates the reCaptcha process whenever one is detected. The extension also extracts the response which can be used on a client computer to bypass captcha instantly.

See Buster on GitHub

 How it works

How does noReCap transfer the solve between computers?

1

Solving

A host computer is pre-programmed to solve reCaptchas for a specific domain.
The response is specific to each domain due to the reCaptcha 'site-key'.

2

Sending the response

After solving the reCaptcha, noReCap takes the 'recaptcha-response' and sends it to the client computer.
The response is not tied to an IP address so the solution can be used on any computer worldwide.

3

Completion

The client can edit the 'recaptcha-response' of the desired website to completely bypass the captcha on the website within milliseconds.

Note: The creator of Buster has publicly expressed that there will be no development for Buster to make it completely automated. NoReCap is not associated with the creator of Buster.
